# Hello World POC

### Usage

1. Compile code
```bash
ghc hello.hs
```

2. Execute code
```bash
./hello
```

3. Output
```haskell
Hello, World!
Here are some odd numbers: [11,13,15,17,19]
```

__Alternative__

A. Load file to `ghci`
```bash
ghci hello.hs
```

B. Call the function via `ghci`
```bash
ghci hello.hs
GHCi, version 9.4.8: https://www.haskell.org/ghc/  :? for help
[1 of 2] Compiling Main             ( hello.hs, interpreted )
Ok, one module loaded.
ghci> main
Hello, World!
Here are some odd numbers: [11,13,15,17,19]
```
